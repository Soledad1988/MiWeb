Detalles del Dashboard ONG

En este caso el protagonista ha sido Looker Studio, facilita la creación de paneles interactivos e informes atractivos a partir de una amplia variedad de fuentes de datos, lo que permite tomar decisiones empresariales más informadas.

  • ETL: limpieza de datos y creación de tablas, utilizando Google Sheets.
  • Segmentación: Implementación de filtros interactivos para análisis por categoría, o años.
  • Navegación entre páginas: Facilita el flujo entre diferentes reportes dentro del dashboard.

Detalles del Dashboard ONG

Creado en Google Sheets, este dashboard muestra las principales funcionalidades para el análisis de datos, incluyendo:

  • Tablas dinámicas: Análisis flexible y dinámico de los datos con opciones de filtrado y resumen.
  • Función QUERY: Unificación de múltiples tablas para realizar análisis más completos.
  • Gráficos: Visualización de tendencias y comparaciones a través de diferentes tipos de gráficos.

Detalles del Dashboard Venta Pizzas

El presente informe proporciona información detallada sobre las ventas de pizzas, utilizando un modelo de datos un poco más complejo y desafiante, conocido como modelo de copo de nieve.

  • Modelado: Modelo copo de nieve.
  • Segmentación: Implementación de filtros interactivos para análisis por categoría, fecha, o productos específicos.
  • Navegación entre páginas: Facilita el flujo entre diferentes reportes dentro del dashboard.
  • Power Query: Limpieza y transformación de datos para optimizar el análisis.
  • DAX: Uso de medidas como DISTINCTCOUNT, SUM, SUMX, MAX, AVERAGE, COUNTROWS.
